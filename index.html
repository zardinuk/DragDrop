DragDrop by kbjr

A JavaScript micro-framework for adding drag-and-drop functionality to elements for advanced UI development

Example

var element = /* ... create element ... */;
var handle = /* ... create element ... */;

var draggable = DragDrop.bind(element, {
    // The anchor; element moves when handle is dragged
    anchor: handle,
    // The draggable element will now stay bound within the window dimensions
    boundingBox: 'windowSize'
});
